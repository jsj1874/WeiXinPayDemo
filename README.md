# WeiXinPayDemo
微信支付 微信扫码支付 微信H5支付

注意：本项目的一些配置文件为需保密，故config文件下的key值为不可信

本项目具体流程
1.微信在线支付简介及申请
2.微信在线支付API简介及测试环境搭建
3.请求支付接口 PC端显示支付二维码
4.异步请求处理notify_url
5.同步跳转return_yrl
6.查询订单接口调用
7.关闭订单接口调用
8.申请退款接口调用
9.查询退款接口调用
10.下载对账单接口调用
11.下载资金账单接口调用
12.微信HTLM5在线支付
